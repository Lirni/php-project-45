## Description
"Brain Games" is a set of five console games based on popular mobile brain-training apps. Each game asks questions that need to be answered correctly. After three correct answers, the game is considered won. If the answer is incorrect, the game ends.

### Games included:
*   **Brain-Even**: Answer "yes" if the number is even, otherwise answer "no".
*   **Brain-Calc**: Solve arithmetic expressions (+, -, *).
*   **Brain-Gcd**: Find the greatest common divisor of two numbers.
*   **Brain-Progression**: Find the missing number in an arithmetic progression.
*   **Brain-Prime**: Answer "yes" if the number is prime, otherwise answer "no".

## Minimum Requirements
*   PHP >= 8.2
*   Composer

## Installation
Clone the repository and install dependencies:

cd php-project-45
make install
